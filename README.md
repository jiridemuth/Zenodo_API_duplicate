Zenodo Duplicator  – User Manual

This tool is designed to rapidly create draft records on the Zenodo (RDM environment) platform using a existing JSON file from Zenodo.
For the program to function correctly, the following files must be located in the same folder as the *.py file:<br>
•	token.txt: A text file containing your personal Zenodo access token.<br>
•	Get yours here: https://zenodo.org/account/settings/applications/ (Scopes: deposit:actions and deposit:write)
